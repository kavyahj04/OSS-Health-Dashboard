# Contributing to OSS Health Dashboard

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/kavyahj04/OSS-Health-Dashboard`
3. Install dependencies: `npm install`
4. Copy env file: `cp .env.example .env.local` and fill in your keys
5. Run locally: `npm run dev`

## Making Changes

- Create a new branch: `git checkout -b feature/your-feature-name`
- Make your changes and commit: `git commit -m "description of change"`
- Push and open a Pull Request

## Code Style

- TypeScript strict mode is enabled — no `any` unless necessary
- Use Tailwind CSS for styling
- Keep components small and focused

## Reporting Issues

Open a GitHub Issue with a clear title and description of the bug or feature request.
